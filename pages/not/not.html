NOT Gate

  • The NOT gate is also called Inverter as it gives the compliment of the input.

  • Output of NOT gate will be 1 (high) when input is 0 (low) and vice versa.

  • It is denoted x=A' in Boolean algebra, which means that x equals NOT A.

  • Standard Symbol for NOT Gate

  • IEC Symbol for NOT Gate

The mathematical formula used for the gate is x=A'.

The Truth table is given as

Input 1 | TRUE

Output | FALSE
