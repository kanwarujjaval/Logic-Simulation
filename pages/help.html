Basic

  • The Infromation about the gates can be accessed by selecting the Gate from home Screen.

Simulations

  • The Simulations are run by clicking on the Input switches

  • The color Red denotes 0, False or LOW value of Input

  • The color Green denotes 1, True or High value of Input

Notations

  • The Symbol ' denotes the compliment or Inverted value of any input

  • In the equations of Logic Gates, x is assumed to be the output of the gate.
