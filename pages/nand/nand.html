NAND Gate

  • The NAND gate operation is performed as multiplication of 1s and 0s and the result is Inverted.

  • Output of NAND gate will be 0 (low) only if all the inputs are 1 (high).

  • It is denoted as x=(A•B)' or x=A|B in Boolean algebra, which means that x equals A NAND B.

  • Standard Symbol for NAND Gate

  • IEC Symbol for NAND Gate

The mathematical formula used for the gate is x=(A•B)', which means, compliment of product of A and B.

The Truth table is given as

Input 1 | TRUE

Input 2 | TRUE

Output | FALSE
