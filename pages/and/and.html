AND Gate

  • The AND gate operation is performed as multiplication of 1s and 0s.

  • Output of AND gate will be 1 (high) only if all the inputs are 1 (high).

  • It is denoted as x=A•B or x=A&B in Boolean algebra, which means that x equals A AND B

  • Standard Symbol for AND Gate

  • IEC Symbol for AND Gate

The Mathematical formula used for the gate is x=A•B

The Truth table is given as

Input 1 | TRUE

Input 2 | TRUE

Output | TRUE
