XNOR Gate

  • The XNOR gate operation is also called Exclusive NOR Gate and it works as inverse of XOR gate.

  • Output of XNOR gate will be 1 (high) when both of the inputs are the same.

  • It is denoted x=(A.B)+(A'.B') in Boolean algebra, which means that x equals A XNOR B.

  • Standard Symbol for XNOR Gate

  • IEC Symbol for XNOR Gate

The mathematical formula used for the gate is x=(A.B)+(A'.B').

The Truth table is given as

Input 1 | TRUE

Input 2 | TRUE

Output | TRUE
